# 🎲 Disider-v1

Disider is a simple web application that helps you make a decision when you have multiple equally good options.

Just enter your choices, click **Pick One**, and let fate decide.

This project was built while I was learning **HTML, CSS, and JavaScript**. It is my first complete frontend project and marks an important milestone in my web development journey.

## ✨ Features

- ➕ Add unlimited choices
- ❌ Remove individual choices
- 🎲 Randomly select one choice
- 💾 Save choices using Local Storage
- 📱 Responsive design for desktop and mobile
- ✨ Animated result card
- 🎉 Floating sparkle animations
- 🚫 Prevent duplicate choices
- 🗑️ Clear all choices with confirmation

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Browser Local Storage API

No frameworks or external libraries were used.

---

## 🚀 How It Works

1. Enter a choice.
2. Press **Enter** or click the **+** button.
3. Repeat until all choices are added.
4. Click **Pick One**.
5. The application randomly selects one option.
6. Your choices remain saved even after refreshing the page.

---

## 💾 Local Storage

Disider uses the browser's **Local Storage**.

This means your choices are stored inside your browser and will still be there after refreshing or reopening the page.

No information is sent to any server.
